# 🌐 3D Portfolio Website

A modern and interactive **3D Portfolio Website** built using **React, Vite, and Three.js**, designed to showcase projects, skills, and contact information in an engaging way.

---

## 🚀 Live Demo

🔗 https://3-d-portfolio-gudalasaivenkat130231-8933s-projects.vercel.app

---

## 📌 Features

* 🎨 Interactive 3D animations using Three.js
* ⚡ Fast performance with Vite
* 📱 Fully responsive design
* 🧩 Modular and reusable components
* 📬 Contact section with email integration
* 🌙 Modern UI/UX design

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Vite
* **3D Library:** Three.js, @react-three/fiber
* **Styling:** CSS / Tailwind CSS
* **Deployment:** Vercel

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/gudalasaivenkat/3_D_portfolio.git
```

2. Navigate to project folder:

```bash
cd reactjs18-3d-portfolio
```

3. Install dependencies:

```bash
npm install
```

4. Run development server:

```bash
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```
